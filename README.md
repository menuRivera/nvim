# Nvim rice
## Installation
1. Install [neovim](https://neovim.io)
2. Clone this repository inside ~/.config/
```sh
git clone https://github.com/menuRivera/nvim
```
3. Open nvim
```bash
nvim
```
4. Let everything install
5. Enjoy
